# 医院预约挂号管理系统

## 主函数 （main.c）

管理系统的入口

存储当前操作用户的操作权限
权限分为以下几种类型：

1. 管理员
2. 医生
3. 患者

## 工具库（utils.h）

包含增、删、改、查等相关操作的函数

* init()
  用于初始化系统
  设置系统编码防止输出乱码
* printDocList()
  打印当前系统已经读取的医生数据
* printResList()
  打印挡墙系统已经读取的患者预约数据
